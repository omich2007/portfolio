Фаренгейт Цельсий

Погодный сайт показывающий текущую погоду в Вашем регионе.

В этом примере, работа основанна на свойствах HTML5 геолокация. Следует отметить, что для правильной работы примера, в браузере должна быть разрешена использование геолокации.

Запрос к API ресурсам написан на "чистом" Java Script. После запроса Json файл переобразуеться в информацию на экране.

В зависимости от погодных условий изменяеться иконка погодных условий и фон body. Реализация основана на простых if ориентированая на влажность воздуха. Конвертор на Фаренгейты основан на простом математическом вычислении.
